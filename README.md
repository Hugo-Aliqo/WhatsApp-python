# WhatsApp-python

WhatsApp-python est une bibliothèque Python qui permet d'envoyer des messages via WhatsApp en utilisant l'interface de WhatsApp Web. Ce projet est conçu pour faciliter l'automatisation de l'envoi de messages, rendant la communication plus efficace.

## Fonctionnalités

- **Envoi Automatisé de Messages** : Envoyez des messages WhatsApp à vos contacts directement depuis un script Python.
- **Configuration Simple** : Ajoutez vos contacts en utilisant un format clé-valeur simple, ce qui rend la configuration rapide et intuitive.
- **Support de Multiples Contacts** : Envoyez des messages à plusieurs contacts en les séparant par des virgules.
- **Aucune Dépendance Externe** : Le projet est léger et ne nécessite pas de bibliothèques tierces complexes.

## Prérequis

- Python 3
- Accès à [WhatsApp Web](https://web.whatsapp.com/) via un navigateur compatible
