# WhatsApp-python
Vous pouvez désormais envoyer des message sur WhatsApp avec Python en passant par votre navigateur.

Rendez vous sur [WhatsApp web](https://web.whatsapp.com/) pour mettre WhatsApp sur votre navigateur. Télécharger le code puis modifier les contacts en mettant le nom du contact suivit de son numéro. Exemple: "Jimmy": 33640475021 si ce n'est pas le dernier contact alors ajouter une virgule puis mettez le contcat suivant.
